Monarch Metal Z-Clip Configurator

How to use the configurator:
  • To see the results on Google Chrome: 3 dots > More Tools > Developer Tools > Console
  • Fill up the required fields (Company Name, Z Clip Option, Quantity, Length, Spacing)
  • Click "Add to cart" button, check answers on Developer Tools Console tab
  • To clear the form data, but leave the input boxes, utilize the "Clear All Fields" button
  • Obs: To reset everything, please reload the page
